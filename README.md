# Django

- Criar pasta vazia
- Criar ambiente virtual
```
python -m venv venv
# or
python3 -m venv venv
```
- Ativar ambiente virtual
```
source venv/bin/activate
```

- Atualizar o pip
```
pip install --upgrade pip
```
- Instale a dependência do Django
```
pip install Django
```
- Inicializar aplicação
```
django-admin startproject meusite .
```
